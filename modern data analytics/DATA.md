# Data Access
 
This project requires the following data files. Download and unzip each into the **root directory** of the repository.
 
| File | Size | Source | Notes |
|------|------|--------|-------|
| `raw.zip` | ~240 MB | [Download from Google Drive](https://drive.google.com/file/d/1ocA_e34dj4ed7PJot-sTxDYfgEJ4VYZS/view?usp=sharing) | Raw aggregated data — required for full pipeline |
| `curated.zip` | — | Included in repo | Pre-processed data for the EDA notebook |
| `ext.zip` | — | Included in repo | External reference files |
 
> **Note:** If you only want to explore the EDA notebook without rerunning the full pipeline, `raw.zip` is optional — the notebook can load from the curated files directly.

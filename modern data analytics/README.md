# Municipality Biking Safety Risk Analysis
This project analyzes municipal safety trends and risk profiles in Flanders.

## Setup Instructions
To run this project, please follow these steps to set up your environment:

 
### 1. Clone the Repository
 
```bash
git clone https://github.com/aradmehralian/Modern-Data-Analytics-Project.git
cd Modern-Data-Analytics-Project
```
 
### 2. Create a Virtual Environment
 
**Windows**
```bash
python -m venv venv
venv\Scripts\activate
```
 
**macOS / Linux**
```bash
python3 -m venv venv
source venv/bin/activate
```
 
### 3. Install Dependencies
 
```bash
pip install -r requirements.txt
```
 
 
## Data Requirements
 
| File type | Location | Notes |
|-----------|----------|-------|
| Curated & external files | Provided as `.zip` archives | Used in the EDA notebook — unzip into their named folders |
| Raw aggregated file | Google Drive | Too large to include in the repo — see [`DATA.md`](DATA.md) for download instructions |
 
> **Tip:** The raw files are optional. The EDA notebook includes steps to load pre-processed files directly, so you can skip the raw data download and pick up from there.
